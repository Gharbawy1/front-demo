حذف بيانات المستخدم

ملخص (بالعربي): يمكنك طلب حذف بيانات حسابك من التطبيق وسنقوم بمعالجة الطلب خلال فترة زمنية معقولة. يرجى اتباع الخطوات أدناه لإرسال طلب حذف البيانات.

خطوات حذف البيانات

  1. أرسل طلب حذف من داخل التطبيق (صفحة الإعدادات → حذف الحساب) أو عبر البريد إلى: your-email@example.com.
  2. عند استلام الطلب سنقوم بالتحقق من هويتك للتأكد من أن صاحب الحساب هو الذي يطلب الحذف.
  3. بعد التحقق سنقوم بحذف بياناتك أو تعطيل حسابك ضمن قواعدنا وسياساتنا. سنؤكد اكتمال الحذف عن طريق بريد إلكتروني أو عبر رابط تأكيد يتم عرضه هنا.
ملاحظة: حذف البيانات يشمل السجلات الشخصية والـ profile المرتبط بحسابك، لكن قد نحتفظ ببعض البيانات المجهولة (aggregated/anonymized) لأغراض قانونية أو إحصائية.

معلومات إضافية

إذا كان طلب الحذف مقدمًا عبر فيسبوك (Data Deletion Request)، قد يستدعي فيسبوك رابط تأكيد (Data Deletion Confirmation URL) أو استجابة JSON من endpoint خاص بنا. لتكامل كامل مع فيسبوك، يجب أن نوفر:

  • رابط Data Deletion Instructions URL (هذه الصفحة) ليعرضها فيسبوك للمستخدم.
  • اختياريًا: Data Deletion Callback endpoint في السيرفر يستجيب لطلبات فيسبوك ويعيد JSON يشرح الحالة أو رابط التأكيد.

مثال (نموذج) لإنشاء Data Deletion Callback على الخادم

إذا أردت أن تنفذ endpoint يتعامل مع طلبات فيسبوك، فمثلاً endpoint مثل:

POST https://your-domain.com/api/privacy/facebook-data-deletion

فيسبوك سترسل body مع معلومات المستخدم (مثال) ويجب أن تعيد استجابة 200 JSON على الشكل التالي (أحد الخيارات):

{
  "url": "https://your-domain.com/privacy/data-deletion-confirmation?user_id=USER_ID",
  "confirmation": "We have scheduled deletion. The data will be removed within 30 days."
}

أو يمكن إرسال استجابة تؤكد الحذف مباشرة إذا تم تنفيذه فورياً:

{
  "status": "deleted",
  "message": "User data deleted successfully."
}

English (short)

This page explains how users can request deletion of their personal data from the application. To request deletion, contact your-email@example.com or use the in-app “Delete Account” action. After identity verification, we will process the deletion and confirm via email or this confirmation URL.

Contact

Questions or deletion requests: your-email@example.com
